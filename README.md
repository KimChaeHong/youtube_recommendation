## 📢 주요기능

- 반응형 페이지
- API를 활용하여 서버에서 동영상 불러오기
- 사이드바 축소 및 확대
- 동영상 정렬 및 검색
- 마우스 오버 동영상 자동 재생
- 동영상 페이지 댓글 추가
- AI를 활용한 관련 동영상 추천

<br />


## 🙌 협업

- 노션, 디스코드를 통해 소통하며 매일 릴레이 코딩 방식으로 진행
- html, css_1, css_2, js_1, js_2 브랜치로 나누어 매일 각자 맡은 역할에 따라 작업
- 매일 작업 완료 후 main브랜치에서 머지 및 팀 노션에 작업 내역 정리
